# TableData Gramplet

This is an experiment on how well an AI service can generate a Gramps addon. I used Claude AI and it was able to generate all code and also documentation and tests.

The idea was to make a gramplet that can store arbitrary tabular data for any object (person, family etc.). However, it is not clear if such a feature is useful for genealogy research. So this is more like an AI coding experiment.

The user documentation generated by Claude is here: [TableData_UserGuide.md](TableData_UserGuide.md).
